== Installation ==

1. Use the automatic installer or unzip and upload the folder wp-jobscore via FTP client to the wp-content/plugins directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Connect your JobScore account to the plugin by inserting your “JobScore Careers Site URL” into the settings page field.
4. Decide which page you want your Careers site to reside on from the dropdown.
5. To choose your formatting options check the “Show Powered by JobScore” box.

== Frequently Asked Questions ==

= Can I use the JobScore plugin if I don’t have a JobScore account? =
No. The JobScore plugin is for top recruiters who use JobScore’s awesome tools. If you don’t yet have a JobScore account please sign up for a free account. http://www.jobscore.com/signup

= Why can’t I get the Advanced Controls to work? =
You must have a Professional or Enterprise account to use these features. You can [upgrade here](http://www.jobscore.com/employer/plans/upgrade?missing_feature=widget_advanced_controls).
  
Basic Controls are available if you are on the Free plan and have checked the "Powered by JobScore" checkbox.
  
= I’ve checked the “Show Powered by JobScore” box and set my widget customization controls, but now the controls aren’t available. =
You must keep the Powered by JobScore checkbox marked to “Show” in order to use the widget customization controls. If you uncheck the box after setting up your controls they will be reset to the default “off” status.
